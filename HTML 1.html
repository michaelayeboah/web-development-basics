The Internet

What is the internet?

The Internet is a global network of computers connected to each other that allows the sharing of information and communication. It connects millions of private, public, academic, business, and government networks using standardized communication protocols.

How does the internet work?

The Internet works by transferring data through a system of routers, servers, and devices using the Transmission Control Protocol/Internet Protocol (TCP/IP). When you send or request information online, your device breaks it into small pieces called packets, which travel through different networks and are reassembled when they reach their destination.

What is a Domain Name?

A domain name is the human-readable address of a website — like www.google.com — that makes it easier to access instead of using the site’s numeric IP address (e.g., 142.250.190.78). Domain names are managed through registrars and organized under extensions like:

  1. .com
  2. .org
  3. .edu

DNS and How it Works

DNS (Domain Name System) is like the phonebook of the Internet. It translates domain names (like www.knust.edu.gh) into IP addresses (like 192.168.1.1), which computers use to identify each other. When you type a website name, your browser asks a DNS server to find the matching IP address so it can load the correct website.

Browsers and How They Work

A web browser (like Chrome, Safari, Firefox, or Edge) is software that allows you to access and view websites. Browsers take the code of a webpage — written in HTML, CSS, and JavaScript — and render it visually on your screen. They use an internal component called a rendering engine to interpret the code and display it as text, images, and interactive elements.

What are Markup Languages?

Markup languages are systems used to structure and format content on web pages. They use tags to define elements like headings, paragraphs, links, and images. The most common markup language is HTML (HyperText Markup Language), which is the backbone of all web pages.

What is Front-End Development?

Front-end development involves building the user interface (UI) and user experience (UX) of websites — the part that users see and interact with. It includes:

  • HTML for structure
  • CSS for styling
  • JavaScript for interactivity
Front-end developers ensure websites look good and function smoothly across devices.

What is HTTP?

HTTP (HyperText Transfer Protocol) is the communication protocol used to transfer data between a web browser and a web server. When you enter a web address, the browser sends an HTTP request to the server, and the server sends back an HTTP response (usually an HTML document). A more secure version is HTTPS, which encrypts data for privacy and protection.

What is a Browser?

A browser is an application that enables users to search, retrieve, and view information from the web. Popular browsers include:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge
  • Safari
  • Opera
They all interpret web code and make it visually accessible.

How do Browsers Work?

Browsers work through the following process:

  1. You enter a URL or search query.
  2. The browser sends an HTTP/HTTPS request to the web server.
  3. The server responds with the website’s code.
  4. The browser’s rendering engine (like Blink or Gecko) processes the HTML, CSS, and JavaScript.
  5. It displays the final webpage for you to view and interact with.

How does the Web Work?

The World Wide Web (WWW) is a collection of information and resources connected through the Internet. It works through three main components:

  • Web Servers: Store and send website files.
  • Web Browsers: Request and display those files.
  • HTTP/HTTPS Protocols: Facilitate communication between browsers and servers.
When you visit a website, your browser communicates with a web server using HTTP, receives HTML content, and displays it as a web page.
